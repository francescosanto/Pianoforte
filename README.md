# Francesco Santo - Portfolio Website

Sito portfolio professionale per pianista.

## Struttura

```
├── index.html          # Pagina principale
├── styles.css          # Stili CSS
├── script.js           # JavaScript
├── assets/
│   ├── audio/          # File audio MP3
│   ├── demo-video.mp4  # Video principale
│   └── video-poster.jpg # Poster video
```

## Setup

1. Aggiungi i file audio nella cartella `assets/audio/`:
   - track1.mp3, track2.mp3, track3.mp3, track4.mp3, track5.mp3

2. Aggiungi il video principale:
   - `assets/demo-video.mp4` (bianco e nero, cinematico)
   - `assets/video-poster.jpg` (immagine di anteprima)

3. Personalizza i contatti in `index.html`:
   - Email
   - Numero di telefono

## Deploy

Il sito è statico e può essere hostato su:
- Netlify
- Vercel
- GitHub Pages
- Qualsiasi hosting web

## Tecnologie

- HTML5
- CSS3 (variabili CSS, Grid, Flexbox)
- JavaScript vanilla
- Google Fonts (Cormorant Garamond, Source Sans 3)
